Native Integration - Easy Video Downloader

To allow the "Easy Video Downloader" extension to send downloading jobs to a native download manager, you need to take an additional step. Please download and install this open-source NodeJS native client by unzipping it first.

  • Click here to download the package for you OS (Window, Mac, and Linux are currently supported) or click here to browse all available download options.
  • Extract the downloaded package in a local directory
  • Now open a terminal in this directory (the directory that contains install.sh and uninstall.sh) and run ./install.sh
  • Now double-click the install.bat file

The installation script for the native client will copy a few files to two different locations so that your browser can detect the client. If you already have NodeJS installed on your system, the installed version will be used. It is important to ensure that you have an up-to-date version of NodeJS installed. If you do not have NodeJS installed, a portable version will be used instead. For more information, visit the andy-portmen/native-client project.

To completely remove this native client, run ./uninstall.shuninstall.bat.

To ensure that the native client is accessible, click on the "Check Connection" button. If you are still seeing this page even when the "Check Button" reports a successful connection, it means the application is not accessible (the path may be incorrect in the options page or the extension's download interface). If the path to the executable is correct and the native client is connected, you can debug the connection by enabling console logs in Chrome, Edge and Opera or Firefox.
